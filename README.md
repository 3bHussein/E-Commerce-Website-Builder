## 📖 Overview

**E-commerce ERP Commerce Suite** is a PHP and MySQL based business platform that combines:

- A modern **responsive e-commerce storefront**
- A complete **ERP command center**
- **B2B and B2C sales workflows**
- **CRM, quotations, invoices, procurement, inventory, finance, HR, and reports**
- A unified system where **website orders and ERP stock/sales remain connected**

It is designed for businesses that want to manage **online product sales**, **company quotations**, **internal ERP operations**, and **customer workflows** from one application.

## 🚀 Installation

### Prerequisites
- PHP 8.0 or higher
- MySQL 5.7 or higher
- Apache/Nginx web server
- PDO MySQL extension enabled
- `mod_rewrite` recommended for Apache
- XAMPP/WAMP/MAMP supported for local development

---

### Step-by-Step Installation

#### 1. Upload the installer
Upload this file to your server:

```text
installer_erp_commerce_suite.php
```

#### 2. Open it in your browser
Example:

```text
https://yourdomain.com/installer_erp_commerce_suite.php
```

#### 3. Complete the setup form
Enter:
- Database host
- Database port
- Database name
- Database username
- Database password
- Admin email and password
- Company/shop details

#### 4. Run the installation
The installer will automatically:
- Create database tables
- Seed demo records
- Generate folders and PHP pages
- Create config file
- Create `.htaccess`
- Build the ERP and storefront structure

#### 5. Log in and access the system

| Area | Path |
|------|------|
| 🌐 Frontend Store | `/index.php` |
| 👑 Admin Dashboard | `/admin/dashboard.php` |
| 🏭 ERP Dashboard | `/admin/erp/dashboard.php` |
| 👤 Customer Dashboard | `/user/dashboard.php` |

---

## ⚠️ Important After Installation

After successful installation:

```text
Delete or rename the installer file.
```

Recommended:

```text
installer_erp_commerce_suite.php → installer_erp_commerce_suite.disabled
```

Leaving an installer file online on a production server is unsafe.
